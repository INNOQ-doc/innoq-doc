---
title: "Jour de la semaine"
api: "GET https://tools.innochat.ch/weekday"
description: "Cette fonction détermine le jour de la semaine correspondant à une date donnée. Par exemple, la date '2024-10-07' retourne le résultat JSON : `{'weekday': 'Monday'}`. Idéal pour les applications nécessitant une logique de dates, des fonctions de calendrier ou une évaluation automatisée des dates."
---

### Paramètres de l’outil

```json
{
  "type": "object",
  "properties": {
    "date": {
      "type": "string",
      "description": "La date au format ISO-8601. Par exemple 2022-06-15 (15 juin 2022)."
    }
  },
  "required": ["date"]
}
```
Include default metadata: False

```python
import requests

url = 'https://tools.innochat.ch/weekday?date=2024-10-10'
headers = {
    'Authorization': 'Bearer <token>'
}

response = requests.get(url, headers=headers)

if response.status_code == 200:
    print("Request successful!")
    print(response.text)
else:
    print("Request failed with status code:", response.status_code)
    print(response.text)

```
