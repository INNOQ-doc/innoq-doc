Dans l’onglet **« Subscriptions »** de votre tableau de bord Whitelabel, après avoir créé ou lié votre compte Stripe, vous pouvez créer des plans payants et des add-ons grâce aux boutons **« Create new plan »** et **« Create new addons »**.

Si vous souhaitez modifier un plan payant ou un add-on après sa création, nous vous recommandons de le faire directement dans Stripe. En effet, de nombreuses complications peuvent survenir – en particulier si des utilisateurs ont déjà souscrit à ce plan ou add-on.

<Frame>
  ![](/images/whitelabel-dash-8.png)
</Frame>

Dans votre compte Stripe, vous pouvez modifier vos **Products** selon vos besoins.  
Comme visible dans l’exemple ci-dessus, vous pouvez changer les champs suivants :

- Nom  
- Description  
- Métadonnées  
- Liste de fonctionnalités (non visible sur la capture d’écran)  

**Important :**  
- **Ne modifiez jamais le champ « Unit label ».**  
- Vous pouvez modifier la valeur entière ou booléenne de chaque ligne de métadonnées pour ajuster les limites de votre plan.  
- **Ne changez jamais les noms des lignes de métadonnées** (par exemple `agents_count`), car l’application a besoin de ces clés exactes pour communiquer avec Stripe. Toute modification rendrait l’application inutilisable.  

Si vous avez besoin d’aide pour modifier vos produits et prix Stripe, contactez-nous.

Lors de la modification des **Prices**, assurez-vous que **chaque prix possède un Lookup-Key**.  
Ces clés **doivent se terminer par « plan » ou « addon »**.  
Exemples :

- `monthly_pro_plan`  
- `token_limit_addon`  

Les prix créés via le tableau de bord contiennent automatiquement ces clés.  
Pour les prix que vous créez directement dans Stripe, vous devez ajouter manuellement les Lookup-Keys.

## Métadonnées des abonnements

Il est possible de modifier les limites d’un abonnement dans Stripe **même après sa création**.  
Gardez à l’esprit que :

- Les modifications affectent **tous les abonnés existants**.  
- Les augmentations de **Message Credits** ne prennent effet qu’au prochain **rafraîchissement mensuel**.  
- Les réductions **ne retirent pas** les crédits déjà attribués aux utilisateurs.  

Voici les champs de métadonnées disponibles que vous pouvez modifier :

| Nom du champ               | Type de valeur      | Description                                                                                                                                                                                        |
| -------------------------- | ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| agents_count               | integer (ex. 5)     | Nombre d’agents autorisés par chatbot                                                                                                                                                             |
| agent_variables_count      | int                 | Nombre de variables qu’un agent peut capturer (total sur l’ensemble du chatbot)                                                                                                                   |
| agent_tags_count           | int                 | Nombre de tags qu’un agent peut attribuer (également totalisé sur l’ensemble du chatbot)                                                                                                          |
| functions_count            | int                 | Nombre de fonctions autorisées par agent                                                                                                                                                           |
| message_credits_replenish  | int                 | Nombre de Message Credits renouvelés chaque mois                                                                                                                                                  |
| message_credits_bonus      | int                 | Nombre de Message Credits offerts à l’inscription (non rétroactif)                                                                                                                                |
| storage_bytes              | int                 | Nombre maximal de bytes pour le stockage des datasources                                                                                                                                          |
| storage_bytes_unlimited    | boolean (ex. True)  | Indique si le stockage est illimité                                                                                                                                                               |
| context_token              | int                 | Nombre de tokens de contexte pour les datasources dans un chatbot                                                                                                                                 |
| max_model_size             | int                 | Taille maximale de contexte disponible pour les modèles                                                                                                                                           |
| chatbots_count             | int                 | Nombre maximal de chatbots par compte                                                                                                                                                             |
| collaborators_count        | int                 | Nombre maximal de collaborateurs par chatbot                                                                                                                                                      |
| overrides_enabled          | bool                | Si l’utilisateur peut utiliser les overrides superviseur                                                                                                                                          |
| dashboard                  | bool                | Si l’utilisateur peut voir le dashboard                                                                                                                                                           |
| visitor_tracking           | bool                | Si l’utilisateur peut voir des statistiques avancées sur les visiteurs (`dashboard` doit être True)                                                                                               |
| gpt_4_enabled              | bool                | Si l’utilisateur peut utiliser des modèles IA avancés (pas seulement GPT-4, mais aussi 4o, etc.)                                                                                                  |
| api_enabled                | bool                | Si l’utilisateur peut utiliser l’API                                                                                                                                                              |
| webhooks                   | bool                | Si l’utilisateur peut utiliser les webhooks                                                                                                                                                       |
| library_access             | bool                | Si l’utilisateur a accès à la « Chatbot Library »                                                                                                                                                 |
| chatbot_retrain            | bool                | Si les datasources peuvent être automatiquement réentraînées                                                                                                                                      |
| watermark_removal          | bool                | Si l’utilisateur peut supprimer le filigrane de son bot                                                                                                                                           |
| chatbot_profile_picture    | bool                | Si l’utilisateur peut définir une image de profil pour son bot                                                                                                                                    |
| custom_domain              | bool                | Si l’utilisateur peut déployer son bot sur un domaine personnalisé                                                                                                                                |
| custom_watermark           | bool                | Si l’utilisateur peut ajouter un filigrane personnalisé (`watermark_removal` doit être True)                                                                                                      |
| user_identity_verification | bool                | Si l’utilisateur peut utiliser la fonction de vérification d’identité                                                                                                                             |
| byok_enabled               | bool                | Si l’utilisateur peut utiliser ses propres clés API IA au lieu des Message Credits                                                                                                                |
| human_escalation_enabled   | bool                | Si l’utilisateur peut intégrer une escalade humaine dans son bot                                                                                                                                  |
| image_upload_enabled       | bool                | Si l’utilisateur peut uploader des images comme datasources                                                                                                                                       |
| allow_chatbot_transfer     | bool                | Si l’utilisateur peut transférer des chatbots à d’autres utilisateurs                                                                                                                             |
| chunk_view_enabled         | bool                | Si l’utilisateur peut voir et modifier directement les chunks extraits des datasources                                                                                                            |
| source_tracking_enabled    | bool                | Si l’utilisateur peut citer et partager directement les sources dans les réponses du chatbot                                                                                                      |
| zapier_integration         | bool                | Si l’utilisateur peut utiliser votre intégration Zapier ou Make.com (les intégrations doivent avoir été publiées au préalable)                                                                     |
| meta_integration           | bool                | Si l’utilisateur peut connecter le bot à Facebook Messenger, Instagram ou WhatsApp (l’app Meta doit être configurée selon les instructions)                                                        |
| sms_enabled                | bool                | Si l’utilisateur peut connecter son bot à des messages SMS (le service Ytel-SMS doit être configuré selon les instructions)                                                                        |
