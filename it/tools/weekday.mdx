title: "Giorno della settimana"
api: "GET https://tools.innochat.ch/weekday"
description: "Questa funzione determina il giorno della settimana corrispondente a una data specificata. Ad esempio, la data '2024-10-07' restituisce il risultato JSON: `{'weekday': 'Monday'}`. Ideale per applicazioni che richiedono logica sulle date, funzioni di calendario o valutazione automatica delle date."
---

### Parametri dello strumento

```json
{
  "type": "object",
  "properties": {
    "date": {
      "type": "string",
      "description": "La data in formato ISO-8601. Ad esempio 2022-06-15 (15 giugno 2022)."
    }
  },
  "required": ["date"]
}
```
Include default metadata: False

```python
import requests

url = 'https://tools.innochat.ch/weekday?date=2024-10-10'
headers = {
    'Authorization': 'Bearer <token>'
}

response = requests.get(url, headers=headers)

if response.status_code == 200:
    print("Request successful!")
    print(response.text)
else:
    print("Request failed with status code:", response.status_code)
    print(response.text)

```
