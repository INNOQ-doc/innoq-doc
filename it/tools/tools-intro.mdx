Quando utilizzi [Function Calling](/rag-from-external-data-provider), di solito hai bisogno di API disponibili pubblicamente – oppure devi crearne di tue. Tuttavia, ospitiamo anche diverse funzioni di tool di uso frequente sul nostro sito: tools.innochat.ch. Ne vengono aggiunte di nuove ogni giorno, quindi tieni d’occhio gli aggiornamenti!

Se desideri suggerire una funzione da aggiungere, faccelo sapere! Puoi contattarci via email all’indirizzo [hello@innochat.ch](mailto:hello@innochat.ch).

Per utilizzare queste funzioni di tool, devi prima creare la tua chiave API INNOCHAT personale. Maggiori dettagli sono disponibili qui: [Ottenere una chiave API INNOCHAT](/api-reference/api-key-setup).

## Utilizzo

Su ogni pagina dedicata a una singola funzione di tool troverai la descrizione e i parametri dello strumento. Questi possono essere utilizzati direttamente quando crei una funzione nel tuo agente. Le impostazioni mostrate nell’immagine qui sotto provengono ad esempio direttamente dalla pagina [Weekday](/tools/weekday).

<Frame>
  <img src="/images/tools-intro-1.png" />
  <img src="/images/tools-intro-2.png" />
</Frame>

Nella sezione «Headers» del modulo devi inserire la chiave API INNOCHAT che hai creato in precedenza, in questo modo:

```json
{
  "Authorization": "Bearer Your-INNOCHAT-API-key-here"
}
```

<Warning> Devi sostituire Your-INNOCHAT-API-key-here con la tua effettiva chiave API. Altrimenti la funzione dello strumento non funzionerà. </Warning>

E poi clicca semplicemente su «Save» e tutto dovrebbe essere pronto!
