---
title: Ottenere una chiave API INNOCHAT
---

La seguente guida ti mostra passo dopo passo come ottenere una chiave API.  
Se possiedi già una chiave API, puoi anche esplorare esempi pratici e suggerimenti di integrazione per iniziare rapidamente con INNOCHAT.  
Iniziamo!

## Passo 1: Accedere alla piattaforma INNOCHAT

Vai sulla [piattaforma INNOCHAT](https://app.innochat.ch).

## Passo 2: Creare un account (se non ne hai ancora uno)

Se non hai ancora un account sulla piattaforma INNOCHAT, clicca su **« Sign Up »** o **« Create Account »**  
e segui il processo di registrazione. Inserisci le informazioni richieste per creare il tuo account.

## Passo 3: Accedere al menu del tuo account

Dopo aver effettuato l’accesso con successo, verrai reindirizzato all’applicazione.  
Cerca nella barra di navigazione la voce **« Account »** e cliccaci sopra per aprire le impostazioni del tuo account.

<Frame>
  <img src="/images/api-key-1.png" />
</Frame>

## Passo 4: Navigare nella sezione delle chiavi API

Nelle impostazioni del tuo account troverai una sezione chiamata **« API Keys »**.  
Lì è presente l’opzione **« Generate new key »** per creare una nuova chiave API.  
Cliccaci sopra per avviare il processo.

<Frame>
  <img src="/images/api-key-2.png" />
</Frame>

## Passo 5: Inserire una descrizione e generare la chiave

Nella finestra **« Create New API Key »** inserisci una descrizione in modo da poter riconoscere in futuro lo scopo di questa chiave.  
Dopo aver inserito la descrizione, clicca su **« Generate »** per creare la chiave API.

<Warning>
  Assicurati di copiare la tua chiave API e di conservarla in un luogo sicuro. Non condividerla mai con altre persone, poiché concede accesso ai servizi dell’API INNOCHAT. Tratta la tua chiave API come una password.
</Warning>

<Frame>
  <img src="/images/api-key-3.png" />
</Frame>

Ecco fatto! Hai richiesto, generato e salvato con successo la tua chiave API INNOCHAT.  
Ora puoi utilizzarla per autenticare le tue richieste e integrare INNOCHAT nelle tue applicazioni, script o progetti.

Se sei pronto per approfondire e scoprire esempi pratici sull’utilizzo dell’API INNOCHAT, visita la sezione **« Live Examples »** nella barra laterale sinistra.
