title: Posteingang-Benachrichtigungen
description: Erfahren Sie, wie INNOCHAT Benachrichtigungen fuer neue Konversationen oder Anfragen an menschlichen Support bereitstellt.
---

INNOCHAT kann so konfiguriert werden, dass automatische Benachrichtigungen ausgegeben werden, wenn eine neue AI-Konversation beginnt oder wenn ein menschlicher Support-Agent angefordert wird.

Bitte stellen Sie zuerst sicher, dass Ihr Browser unserer App das Senden von Benachrichtigungen erlaubt.

## Wie behebe ich Probleme mit den Browser-Benachrichtigungseinstellungen?

# Google Chrome

1. Gehen Sie zu https://app.innochat.ch/
2. Klicken Sie auf das Symbol links neben der URL.
3. Stellen Sie sicher, dass die Benachrichtigungen auf **Ask** oder **Allow** gesetzt sind.

# Mozilla Firefox

1. Gehen Sie zu https://app.innochat.ch/
2. Klicken Sie auf das Symbol links neben der URL.
3. Stellen Sie sicher, dass die Berechtigung **Send Notifications** nicht blockiert ist.

# Safari

1. Gehen Sie zu Safari > Preferences.
2. Oeffnen Sie den Tab **Websites**.
3. Stellen Sie sicher, dass die Berechtigung fuer Benachrichtigungen fuer https://app.innochat.ch/ auf **Allow** gesetzt ist.

Als naechstes gehen Sie zum Tab **Inbox**.
