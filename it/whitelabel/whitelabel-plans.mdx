Nella scheda **« Subscriptions »** del tuo dashboard Whitelabel, dopo aver creato o collegato il tuo account Stripe, puoi creare piani a pagamento e add-on utilizzando i pulsanti **« Create new plan »** e **« Create new addons »**.

Se desideri modificare un piano a pagamento o un add-on dopo la creazione, ti consigliamo di farlo direttamente in Stripe. Questo perché possono sorgere numerose complicazioni – specialmente se alcuni utenti hanno già sottoscritto quel piano o add-on.

<Frame>
  ![](/images/whitelabel-dash-8.png)
</Frame>

Nel tuo account Stripe puoi modificare i tuoi **Products** secondo necessità.  
Come visibile nell’esempio sopra, puoi modificare i seguenti campi:

- Nome  
- Descrizione  
- Metadati  
- Lista delle funzionalità (non visibile nello screenshot)  

**Importante:**  
- **Non modificare mai il campo « Unit label ».**  
- Puoi modificare il valore intero o booleano di ogni riga di metadati per adattare i limiti del piano.  
- **Non cambiare mai i nomi delle righe di metadati** (ad esempio `agents_count`), poiché l’app richiede esattamente queste chiavi per comunicare con Stripe. Qualsiasi modifica renderebbe l’app non funzionante.  

Se hai bisogno di supporto per modificare i tuoi prodotti e prezzi Stripe, contattaci.

Quando modifichi i **Prices**, assicurati che **ogni prezzo abbia un Lookup-Key**.  
Queste chiavi **devono terminare con « plan » o « addon »**.  
Esempi:

- `monthly_pro_plan`  
- `token_limit_addon`  

I prezzi creati tramite il dashboard includono automaticamente queste chiavi.  
Per i prezzi che crei direttamente in Stripe, devi aggiungere manualmente i Lookup-Keys.

## Metadati degli abbonamenti

È possibile modificare i limiti di un abbonamento in Stripe **anche dopo la sua creazione**.  
Tieni presente che:

- Le modifiche influenzano **tutti gli abbonati esistenti**.  
- Gli aumenti di **Message Credits** diventano effettivi solo al prossimo **refresh mensile**.  
- Le riduzioni **non rimuovono** i crediti già assegnati agli utenti.  

Di seguito i campi di metadati disponibili che puoi modificare:

| Nome campo                 | Tipo valore         | Descrizione                                                                                                                                                                                        |
| -------------------------- | ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| agents_count               | integer (es. 5)     | Numero di agenti consentiti per chatbot                                                                                                                                                           |
| agent_variables_count      | int                 | Numero di variabili che un agente può catturare (totale sull’intero chatbot)                                                                                                                      |
| agent_tags_count           | int                 | Numero di tag che un agente può assegnare (anch’esso sommato sull’intero chatbot)                                                                                                                 |
| functions_count            | int                 | Numero di funzioni consentite per agente                                                                                                                                                          |
| message_credits_replenish  | int                 | Numero di Message Credits rinnovati ogni mese                                                                                                                                                     |
| message_credits_bonus      | int                 | Numero di Message Credits assegnati all’iscrizione (non retroattivo)                                                                                                                              |
| storage_bytes              | int                 | Numero massimo di byte per lo storage delle datasources                                                                                                                                           |
| storage_bytes_unlimited    | boolean (es. True)  | Indica se lo storage è illimitato                                                                                                                                                                 |
| context_token              | int                 | Numero di token di contesto per le datasources in un chatbot                                                                                                                                      |
| max_model_size             | int                 | Dimensione massima di contesto disponibile per i modelli                                                                                                                                          |
| chatbots_count             | int                 | Numero massimo di chatbots per account                                                                                                                                                            |
| collaborators_count        | int                 | Numero massimo di collaboratori per chatbot                                                                                                                                                       |
| overrides_enabled          | bool                | Se l’utente può utilizzare gli override supervisore                                                                                                                                               |
| dashboard                  | bool                | Se l’utente può visualizzare il dashboard                                                                                                                                                         |
| visitor_tracking           | bool                | Se l’utente può vedere statistiche avanzate sui visitatori (`dashboard` deve essere True)                                                                                                         |
| gpt_4_enabled              | bool                | Se l’utente può utilizzare modelli IA avanzati (non solo GPT-4, ma anche 4o ecc.)                                                                                                                 |
| api_enabled                | bool                | Se l’utente può utilizzare l’API                                                                                                                                                                  |
| webhooks                   | bool                | Se l’utente può utilizzare i webhooks                                                                                                                                                             |
| library_access             | bool                | Se l’utente ha accesso alla « Chatbot Library »                                                                                                                                                   |
| chatbot_retrain            | bool                | Se le datasources possono essere automaticamente ritrained                                                                                                                                        |
| watermark_removal          | bool                | Se l’utente può rimuovere il watermark dal suo bot                                                                                                                                                |
| chatbot_profile_picture    | bool                | Se l’utente può impostare un’immagine di profilo per il suo bot                                                                                                                                   |
| custom_domain              | bool                | Se l’utente può pubblicare il bot su un dominio personalizzato                                                                                                                                    |
| custom_watermark           | bool                | Se l’utente può aggiungere un watermark personalizzato (`watermark_removal` deve essere True)                                                                                                     |
| user_identity_verification | bool                | Se l’utente può utilizzare la funzione di verifica dell’identità                                                                                                                                  |
| byok_enabled               | bool                | Se l’utente può utilizzare le proprie chiavi API IA invece dei Message Credits                                                                                                                    |
| human_escalation_enabled   | bool                | Se l’utente può integrare un’escalation umana nel suo bot                                                                                                                                         |
| image_upload_enabled       | bool                | Se l’utente può caricare immagini come datasources                                                                                                                                                |
| allow_chatbot_transfer     | bool                | Se l’utente può trasferire chatbots ad altri utenti                                                                                                                                               |
| chunk_view_enabled         | bool                | Se l’utente può visualizzare e modificare direttamente i chunk estratti dalle datasources                                                                                                         |
| source_tracking_enabled    | bool                | Se l’utente può citare e condividere direttamente le fonti nelle risposte del chatbot                                                                                                            |
| zapier_integration         | bool                | Se l’utente può utilizzare la tua integrazione Zapier o Make.com (le integrazioni devono essere state precedentemente pubblicate)                                                                   |
| meta_integration           | bool                | Se l’utente può collegare il bot a Facebook Messenger, Instagram o WhatsApp (l’app Meta deve essere configurata secondo le istruzioni)                                                             |
| sms_enabled                | bool                | Se l’utente può collegare il bot a messaggi SMS (il servizio Ytel-SMS deve essere configurato secondo le istruzioni)                                                                               |
